# 01 — Project Overview
**Vidya Setu: Digital Learning Platform for Rural School Students in Nabha**

---

## Problem Statement
Rural schools in Nabha (Punjab, India) face:
- Outdated digital infrastructure
- Poor or intermittent internet connectivity (2G/3G environments)
- Lack of learning devices (students use basic 4GB RAM Android phones)
- Teachers have no tools to track progress or personalize support
- Students fall behind and miss 21st-century digital skills

**Source:** SIH2025 Problem Statement ID `SIH25019`

---

## Vision
> Pioneer an **"Edge-to-Cloud Synchronized Learning Ecosystem"** — a local-first, peer-to-peer mesh networking layer on mobile devices enabling near-real-time collaborative learning and progress synchronization between students and teachers within school premises (the Edge), **independent of the main internet grid.**

---

## Core Goals
1. Deliver real-time digital classrooms and live chat between teachers and students
2. Support offline learning when connectivity is weak or absent
3. Provide multilingual lessons (English, Punjabi, Hindi)
4. Include teacher dashboards, analytics, and progress tracking
5. Be lightweight and optimized for 4GB RAM Android devices
6. Guarantee **zero-loss offline progress tracking** with background sync to cloud

---

## Users
| Role | Device | Connectivity |
|------|--------|-------------|
| Student | 4GB RAM Android smartphone | 2G/3G/offline |
| Teacher | Android phone or laptop | Intermittent broadband |
| Admin | Desktop/laptop | Stable broadband |

---

## Expected Outcomes
- **Product:** Offline digital learning app with local language lessons, teacher tools, and improved digital literacy
- **Impact:** Improve digital literacy among rural students by 60% within pilot phase
- **Scale:** Enable teachers to track real-time learning of 100+ students simultaneously
- **Access:** Promote bilingual/multilingual learning accessibility

## Constraints
- UI must never crash or lag on 4GB RAM Android
- All core features must work without internet
- Data must sync automatically when connectivity returns — zero loss
- Must support Punjabi, Hindi, and English interfaces
- Must work as a PWA (installable from browser, no app store needed)
